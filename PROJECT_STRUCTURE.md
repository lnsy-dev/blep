```
├── .gitignore                # Git ignore patterns
└── PROJECT_STRUCTURE.md      # This file
```

## File Descriptions

### Core Files

**BLParser.php** (4.3 KB)
- Parses PHP files for `@bl-*` tags
- Tracks topics, subtopics, details, and cross-references
- Handles both docblock and inline comments
- Generates warnings for malformed documentation
- Public API: `addFile()`, `addDirectory()`, `getData()`, `getWarnings()`

**BLHtmlGenerator.php** (4.4 KB)
- Generates static HTML documentation site
- Embedded CSS for self-contained output
- Creates index page and individual topic pages
- Handles cross-references between topics
- Public API: `__construct()`, `generate()`

**bl-doc-gen.php** (6.1 KB)
- CLI entry point for development
- Argument parsing and validation
- Orchestrates parser and generator
- Prints statistics and warnings
- Requires BLParser.php and BLHtmlGenerator.php

**bldoc** (15 KB)
- Single-file distributable version
- Contains all classes bundled together
- No external dependencies or require statements
- Generated by build.sh

### Build & Test

**build.sh** (1.2 KB)
- Bash script to create single-file distributable
- Strips PHP tags and combines all classes
- Makes output executable
- Reports file size

**test.sh** (1.7 KB)
- Automated test suite
- Tests help, version, generation, verbose mode, exclude patterns
- Verifies output files are created correctly
- Tests both development and single-file versions

**example.php** (2.2 KB)
- Sample PHP file with BL tags
- Demonstrates all tag types
- Used for testing and documentation

### Documentation

**README.md** (7.2 KB)
- Complete project documentation
- Installation instructions

- Tag reference
- Tips and best practices
- MIT license

**QUICKSTART.md** (2.5 KB)
- Quick start guide for new users

**CHANGELOG.md**

## Usage Workflow

### Development Mode
```bash
php bl-doc-gen.php [options] <source-path>
```

### Single-File Mode
```bash
./build.sh          # Build once
./bldoc [options] <source-path>
```

### Testing
```bash
./test.sh           # Run all tests
```

## Distribution

For end users, distribute only:
- `bldoc` (single-file version)
- `README.md`
- `QUICKSTART.md` (optional)

The single-file version is completely self-contained and requires only PHP 7.4+.
