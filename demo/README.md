# Blep Demo - E-Commerce Business Rules

This directory contains example documentation generated by Blep from the annotated PHP code in `example/src/`.

## What's Inside

This demo showcases Blep's capabilities by documenting business rules for a fictional e-commerce system:

- **Order Processing** - Order validation, approval workflows, inventory management
- **Payment Processing** - Payment methods, fraud detection, refund policies
- **Shipping** - Shipping methods, cost calculation, delivery rules
- **Returns Policy** - Return eligibility, refund processing, restocking
- **Discount Rules** - Coupon codes, promotional discounts, loyalty programs
- **Customer Accounts** - Account creation, verification, tier management

## Features Demonstrated

- ✅ Topic and subtopic organization
- ✅ Business rule details with source references
- ✅ Rationale explanations (collapsible "Why?" sections)
- ✅ Cross-references between related topics
- ✅ Git integration showing change history
- ✅ Interactive search functionality
- ✅ Timeline changelog of all changes
- ✅ Code snippets in collapsible sections
- ✅ Markdown versions of all documentation

## How It Was Generated

```bash
./bin/bldoc -o demo -t "Blep Demo - E-Commerce Business Rules" example/src/
```
