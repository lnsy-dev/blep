Blep

Document your business logic directly in code with simple @bl-* tags

curl -O https://github.com/yourusername/blep/releases/latest/download/bldoc && chmod +x bldoc

Quick Start

Add tags to your PHP code: 
/**
 * @bl-topic Order Processing
 * @bl-detail Orders over $1000 require manager approval
 */
Run the generator: 
./bldoc src/
Browse your documentation: 
open bl-docs/index.html

Features

Zero Dependencies

Pure PHP 7.4+, no external libraries required. Single-file distribution available.

Simple Syntax

Use @bl-topic, @bl-subtopic, @bl-detail, and @bl-see tags in comments.

Clean Output

Generates responsive, readable HTML documentation with cross-references and source links.

Fast Scanning

Recursively processes directories, respects exclusion patterns, shows progress with verbose mode.

Why Blep?

  • Document business rules where they're implemented
  • Keep documentation in sync with code changes
  • Generate browsable sites for stakeholders
  • Cross-reference related business logic
  • No complex setup or configuration
